# 🎆 Fireworks & Crackers E-Commerce Website

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) e-commerce platform for selling fireworks and crackers online.

## 🚀 Quick Summary

| Item             | Detail                          |
|------------------|---------------------------------|
| **Project Type** | E-Commerce Website              |
| **Niche**        | Fireworks & Crackers            |
| **Stack**        | MERN (MongoDB, Express, React, Node) |
| **Auth**         | JWT + Refresh Tokens            |
| **Payment**      | Razorpay / Stripe               |
| **Hosting**      | Frontend → Vercel / Netlify, Backend → Render / Railway / AWS EC2 |
| **Database**     | MongoDB Atlas (Cloud)           |
| **Storage**      | Cloudinary (Product Images)     |
| **Domain**       | GoDaddy / Namecheap             |

## 📌 Key Features

- 🛒 Product listing, search, filter by category/price
- 🔥 Featured & trending crackers section
- 👤 User registration, login, profile
- 📦 Cart, Wishlist, Order management
- 💳 Secure online payment
- 📍 Address management & delivery tracking
- 🛡️ Admin dashboard (inventory, orders, users)
- 📱 Fully responsive (Mobile first)
- 🔔 Email notifications (Order confirm, dispatch, delivery)
